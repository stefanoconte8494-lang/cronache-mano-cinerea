# Cronache della Mano Cinerea - progetto React

Questo progetto React/Vite contiene una prima wiki narrativa della Mano Cinerea, costruita nello stile della schermata che ti piaceva e popolata con i dati estratti dal file Excel.

## Come avviarlo

1. Apri la cartella in Visual Studio Code.
2. Apri il terminale.
3. Esegui:

```bash
npm install
npm run dev
```

4. Apri l'indirizzo indicato, di solito:

```text
http://localhost:5173/
```

## Dove sono i dati

I dati importati dall'Excel sono in:

```text
src/data/campaignData.js
```

Sono già presenti:
- personaggi
- sessioni
- PNG
- luoghi
- quest
- oggetti
- fazioni
- timeline
